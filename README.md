📚 Smart Assignment Tracker (StudyIQ)
🚀 Project Overview:

The Smart Assignment Tracker (StudyIQ) is a Python-based web application built using Streamlit.
It helps students manage assignments efficiently with login authentication, OTP verification, assignment uploads, and tracking features.
👨‍💻 Features
🔐 Authentication System
User Signup & Login
Email-based OTP verification

👨‍💻 Admin Dashboard
Upload assignments for different subjects
Store files in system

🎓 Student Dashboard
View assignment notifications
Download assignments
Subject-wise organization

⏱️ Smart Tracking
Live assignment statistics
Countdown timer for deadlines

🛠️ Technologies Used
Python
Streamlit
SMTP (Email OTP system)
HTML/CSS (for UI styling)

⚙️ Installation & Setup
1. Clone the Repository
git clone https://github.com/aaryadalvi0807/Python-Miniproject-Remote-.git

2. Navigate to Project Folder
cd Python-Miniproject-Remote-

3. Install Dependencies
pip install streamlit

5. Run the Application
streamlit run app.py

▶️ How It Works

1.User registers using username, email, and password

2.Login with credentials

3.OTP is sent to registered email

4.After verification:
.Admin uploads assignments
.Students receive notifications

5.Students can:
.View subjects
.Download assignments
.Track pending tasks
